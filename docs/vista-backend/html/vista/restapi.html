Status codes¶

Return values

Description

200 OK

The GET request was successful, the resource(s) itself is returned as JSON.

201 Created

The POST or PUT request was successful and the resource is returned as JSON.

204 Deleted

The DELETE request was successful.

400 Bad Request

A required attribute of the API request is missing, e.g., the host_ip of an docker host is not given.

404 Not Found

A resource could not be accessed, e.g., an ID for a resource could not be found.
